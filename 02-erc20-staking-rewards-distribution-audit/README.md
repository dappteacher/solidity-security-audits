# ERC20 Staking Rewards Distribution Audit

This repository contains a security analysis of the following project:

🔗 https://github.com/luzzif/erc20-staking-rewards-distribution-contracts

---

## 📌 Overview

This audit reviews a staking rewards distribution system based on:

* Clone (minimal proxy) pattern
* Multi-reward token support
* Time-based reward emission

The system includes:

* `ERC20StakingRewardsDistribution`
* `ERC20StakingRewardsDistributionFactory`

---

## 🎯 Objectives

* Identify security vulnerabilities
* Evaluate contract design and architecture
* Detect economic and logical risks
* Provide actionable mitigation strategies
